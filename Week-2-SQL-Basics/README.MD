# SQL Sales Database Assignment

# Overview
This project contains SQL queries for a Sales Management Database. It demonstrates database creation, data insertion, filtering, aggregation, joins, conditional logic, indexing, and transaction management using MySQL.

## Tools Used
- MySQL
- MySQL Workbench

## Database Tables
- Customers
- Products
- Orders
- Order_Items

## Topics Covered
- SQL Basics (SELECT, WHERE, DISTINCT)
- Aggregate Functions
- GROUP BY & HAVING
- Joins
- CASE Statements
- Primary & Foreign Keys
- Indexes
- ACID Properties
- Transactions (COMMIT & ROLLBACK)
